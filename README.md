# Blog Website

This is a Frontend project that serves Blogs from my other project [blogBackend](https://github.com/ibrahimwalle/blogBackend)!

## Technologies

* Angular 13
* Responsive Web Design
* Flexbox
* CSS Grid
* Bootstrap
